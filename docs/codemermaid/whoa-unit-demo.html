whoa · code

The reducer adds memory without changing the protocol shape.

The stream needs private identity hints to merge live events with later snapshots. Instead of mutating the SDK schema, chat-kit hides that memory behind a non-enumerable Symbol. The UI gets better dedupe behavior, and external callers still see clean SDK messages.

The composer is one input, but many product controls.

whoa · ux

Tool results update only the renderer waiting for them.

ToolResultProvider keeps a stable external store and lets renderers subscribe by tool_use_id. A late tool result can wake the exact tool card that needs it without making the whole historical transcript repaint.

whoa · architecture

ChatPanel builds a transcript tree, then chooses the cheapest first paint.

Completed history can render as a collapsed transcript first, then hydrate hidden internals on idle or expand. That keeps replay/read-only sessions responsive while preserving the full turn details when the reader asks for them.
